## How to run
```bash
# Clone Repository
$ git clone https://github.com/marcusv77/nextjs-template.git
```
### Run Web Project

```bash
# Go to web folder
$ cd nextjs-template

# Install Dependencies
$ yarn install or npm install

# Run Aplication
$ yarn dev or npm dev
```
Go to http://localhost:3000/ to see the result.
